## 🖼️ Using BrandFit for your projects

The application interface allows you to upload many images at once. The tool detects the shapes and colors of your logos to adjust them to a uniform size.

### Batch processing images
Select a folder or drag and drop your files into the main window. BrandFit organizes these files into a clean grid format. You can choose to resize, crop, or add padding to every image simultaneously.

### Trimming whitespace
Logos often come with uneven borders. The software automatically detects the edges of your logo and removes empty space. This ensures every asset aligns perfectly when you place them side by side.

### Padding and normalization
Consistency matters in design. Use the padding tools to add equal space around every logo. This makes your final output look professional and neat when you add it to websites or documents.

## 📂 Exporting your assets

Once you complete your edits, export your files in the format you need. The tool supports standard image types including PNG and WebP. Simply select the export button to save the result to a new folder on your machine.

## 🛡️ Privacy and data security

We built BrandFit with privacy in mind. Everything happens inside your browser. No files get uploaded to a cloud server or a remote database. The software relies on the processing power of your computer to handle images. When you close the app, all temporary data clears from your browser memory.

## 🛠️ Frequently asked questions

### Will this software slow down my computer?
BrandFit uses your computer memory to process images. If you load hundreds of high-resolution files, you might notice a brief slowdown. We recommend processing files in batches of fifty to keep performance fast.

### Can I change the file format?
Yes. During the export process, you can toggle between PNG and WebP formats. WebP is better for smaller file sizes, while PNG works best for images that need a transparent background.

### What happens if the app crashes?
The application saves your progress in your browser cache. If you accidentally close the window, reload the page. Your current session state should remain intact. If you encounter a persistent error, restart the app and clear your cache.

### Does this support vector files?
The current version focuses on image files like PNG and JPEG. Support for SVG files is in development for future versions. 

### Can I run this offline?
Yes. Because the app runs in your browser using local resources, you do not need an active internet connection to process your images.
